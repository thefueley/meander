# meander

Meander Pattern using Snap SVG

## Setup Virtual Env

`python -m venv venv`

`source venv/bin/activate`

## Install Project Requirements

`pip install -r requirements.txt`

## Setup App

`export FLASK_APP=app`

## Start App

`flask run`

## Snap SVG Homepage

[SnapSVG](http://snapsvg.io/)
